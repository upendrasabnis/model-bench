| 105 | Nonprofits rely on external funding to translate mission‑driven ideas into tangible impact. |

| 109 | **2.1. Types of Funders** |

| 111 | • **Government Agencies** – Federal, state, and local entities issue RFPs for programs that align with public policy goals. |
| 112 | • **Corporations** – CSR (Corporate Social Responsibility) programs may fund projects that align with brand values or employee engagement. |
| 113 | • **Individual Donors & Trusts** – Major gifts can be earmarked for specific initiatives when presented compellingly. |
| 114 | **2.2. Funding Cycles & Deadlines** |
| 115 | • Most foundations operate on an annual cycle (e.g., Letter of Inquiry in January, full proposal in March). |
| 116 | • Government RFPs often have fixed submission windows that can be as short as 30 days. |
| 117 | • Corporate grants may align with fiscal years, creating predictable “grant windows.” |
| 118 | • Build a master calendar to track all relevant deadlines and avoid missed opportunities. |
| 119 | **2.3. Trends in Philanthropy** |
| 120 | • **Impact‑First Funding** – Donors increasingly demand measurable outcomes. |
| 121 | • **Trust‑Based Philanthropy** – Emphasis on unrestricted funding and reduced reporting burden. |
| 122 | • **Equity‑Focused Grants** – Prioritization of projects that address systemic inequities. |
| 123 | • Staying current with these trends helps you position your proposal to meet evolving expectations. |

| 152 | **5. Aligning Goals, Objectives, and Outcomes** |
| 153 | **5.1. SMART Objectives Formula** |
| 154 | • **Specific** – Clearly state what will be achieved. |
| 155 | • **Measurable** – Define how you will know it’s achieved. |
| 156 | • **Achievable** – Ensure resources and capacity exist. |
| 157 | • **Relevant** – Connect to the funder’s priorities. |
| 158 | • **Time‑bound** – Set a clear deadline. |
| 159 | **5.2. Outcome vs. Output Distinctions** |
| 160 | • **Outputs** – Tangible products (e.g., number of workshops delivered). |
| 161 | • **Outcomes** – Changes in knowledge, behavior, or condition (e.g., increase in participants’ self‑efficacy). |
| 162 | • **Impact** – Long‑term transformation (e.g., reduced poverty rates). |
| 163 | **5.3. Logic Model Overview** |
| 164 | • Visual map linking inputs → activities → outputs → outcomes → impact. |
| 165 | • Helps reviewers see the causal pathway at a glance. |
| 166 | **5.4. Building a Logic Model (Step‑by‑Step)** |
| 167 | 1. Identify **Inputs** (staff, funding, partnerships). |
| 168 | 2. Define **Activities** (training, outreach, service delivery). |
| 169 | 3. List **Outputs** (e.g., “30 workshops conducted”). |
| 170 | 4. Specify **Short‑Term Outcomes** (e.g., “80 % of participants improve dietary knowledge”). |
| 171 | 5. Project **Intermediate Outcomes** (e.g., “15 % reduction in household food‑insecurity”). |
| 172 | 6. Articulate **Long‑Term Impact** (e.g., “Improved community health indices”). |
| 173 | **5.5. Example Logic Model for a Youth Mentoring Program** |
| 174 | *Inputs:* Grants, volunteer mentors, curriculum. |
| 175 | *Activities:* Recruit mentors, match with youth, weekly check‑ins. |
| 176 | *Outputs:* 120 mentor‑youth pairs, 24 workshops. |
| 177 | *Short‑Term Outcomes:* 90 % of youth report increased self‑esteem. |
| 178 | *Intermediate Outcomes:* 30 % improvement in school attendance. |
| 179 | *Impact:* 10 % decrease in dropout rates after 2 years. |
