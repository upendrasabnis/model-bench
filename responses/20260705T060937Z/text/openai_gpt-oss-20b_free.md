## 3. Step‑by‑Step Guide to Preparing a Competitive CDBG Application

| Step | Action | Deliverable |
|------|--------|-------------|
| **1. Form a CDBG Task Force** | • Town council, şu/City Manager, Economic Development Director, Housing Coordinator, and community volunteers. | • Charter, meeting schedule. |
| **2. Conduct a Community Needs Assessment** | • Demographic data, economic indicators, infrastructure gaps, housing stock. | • Needs assessment report. |
| **3. Identify Priority Projects** | • Infrastructure upgrades, affordable‑housing rehab, small‑business incubation, workforce training. | • Ranked project list. |
| **4. Draft the CDBG Action Plan** | • Align with the town’s comprehensive plan and state goals. | • Draft action plan (PDF). |
| **5. Secure Matching Funds** | • Identify local, state, or private partners. | • Matching‑fund commitment letters. |
| **6. Perform Cost‑Benefit Analysis** | • Estimate economic impact, job creation, tax revenue. | • Cost‑benefit spreadsheet. |
| **7. Prepare the Application Narrative** | • Address HUD’s “Project Design” and “Project Implementation” sections. | • Draft narrative. |
| **8. Create a Detailed Budget** | • Capital, operational, contingency. | • Budget template (see Section 6). |
| **9. Develop a Timeline Gantt Chart** | • Phases: Planning, Design, Construction, Evaluation. | • Gantt chart (textual). |
| **10. Conduct Stakeholder Consultation** | • Public hearings, business roundtables, neighborhood association meetings. | • Minutes, feedback summaries. |
| **11. Finalize Application** | • Review by HUD compliance specialist, legal counsel. | • Completed application. |
| **12. Submit & Track** | • Submit via HUD’s web portal; monitor status. | • Tracking log. |

### 3.1 Tips for a Competitive Application

- **Data‑Driven Narrative**: Embed hard numbers (population decline, unemployment rate, median income) to justify need.
- **Show Partnerships**: Highlight existing collaborations (e.g., local university, Chamber of Commerce).
- **Demonstrate Sustainability**: Outline how the project will be maintained post‑grant.
- **Use Clear Language**: Avoid jargon; make the narrative accessible to both HUD reviewers and community members.

## 4. Matching Funds, Cost‑Benefit Analysis, and Stakeholder Engagement

### 4.1 Matching Funds

| Matching Type | Example | Pros | Cons |
|---------------|---------|------|------|
| **Cash** | Town’s own budget | Quick, simple | Can strain municipal finances |
| **In‑Kind** | Volunteer labor, donated materials | Reduces cash requirement | Requires coordination |
| **Private‑Sector** | Local business investment | Encourages local buy‑in | May need profit‑sharing agreements |

**Key Rule**: Matching must be *committed* before the grant is awarded.

### 4.2 Cost‑Benefit Analysis (CBA)

| Metric | Description | Sample Target |
|--------|-------------|----------------|
| **Net Present Value (NPV)** | Present value of benefits minus costs | +$2 M |
| **Internal Rate of Return (IRR)** | Discount rate at which NPV=0 | > 12 % |
| **Job Creation** | Direct + indirect jobs over 5 yr | 150 jobs |
| **Tax Revenue Increase** | Additional property and sales taxes | $200 k/yr |
| **Housing Affordability Index** | % of households paying < 30 % income | 12 % improvement |

| **Housing Authorities** | Affordable units, compliance | Joint workgroups, data sharing |
