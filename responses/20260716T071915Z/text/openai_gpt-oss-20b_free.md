## 4. Assessment & Scoring

### 4.1 The Decision‑Making Gap

- **Subjective scoring** can lead to inconsistent decisions.
- **Review panels** often spread across time zones.
- **Traceability** of decisions is limited.

### 4.2 GMS Features That Close the Gap

| Feature | Benefit | Example |
|---------|---------|---------|
| **Automated Eligibility Checks** | Pre‑filters applications before review | *Global Education* automatically excludes proposals with missing budget items. |
| **Structured Scoring Rubrics** | Standardizes criteria across all reviewers | *Health for All* uses a 1–5 scale for impact, feasibility, sustainabilityének. |
| **Blind Review Workflow** | Removes applicant identifiers during scoring | *Climate Action Fund* ensures reviewers only see project titles. |
| **Decision Dashboard** | Visualizes scores, trends, and reviewer fatigue | *Youth Empowerment* monitors reviewer distribution. |
| **Analytics Reports** | Detects bias patterns | *Riverbank Fund* identified a gender bias that was corrected. |

### 4.3 Example Workflow

1. **Automated Eligibility** – 1,200 applications pass the filter.
2. **Peer Review** – 4 reviewers assign scores; system auto‑calculates averages.
3. **Committee Review** – Final decision is logged, with justifications.
4. **Notification** – Winners receive a personalized email; losers get constructive feedback.
