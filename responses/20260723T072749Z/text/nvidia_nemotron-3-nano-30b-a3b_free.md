**8. Checklists for Each Stage**  

**8.1. Pre‑Research Checklist**  
- [ ] Mission alignment confirmed  
- [ ] Funder priority language captured  
- [ ] Recent grant examples reviewed (≥ 3)  
- [ ] Decision‑maker contact info verified  

**8.2. Writing Checklist**  
- [ ] Needs statement includes data, stakeholder quotes, and funder language  
- [ ] Theory of change is logical and measurable  
- [ ] All objectives are SMART  
- [ ] Evaluation plan includes baseline, process, and outcome measures  
- [ ] Budget totals match requested amount  
- [ ] Indirect cost rate applied correctly  

**8.3. Submission Checklist**  
- [ ] All required attachments included (IRS letter, budget, bios)  
- [ ] PDF format meets file‑size limits  
- [ ] Submit via correct portal (Grants.gov, Foundation Online)  
- [ ] Confirmation email received  

**8.4. Post‑Award Checklist**  
- [ ] Award letter filed and archived  
- [ ] Initial reporting schedule set (30‑day, 6‑month)  
- [ ] Quarterly financial reconciliation template ready  
- [ ] Monitoring system (data collection tools) operational  

**9. Common Pitfalls & How to Avoid Them**  
9.1. Misreading RFP Instructions (skipping eligibility clause)  
9.2. Over‑Promising Outcomes (unrealistic metrics)  
9.3. Vague Budget Language (“general operating” without justification)  
9.4. Ignoring Indirect Cost Policies (under‑reporting)  
9.5. Submitting After Deadline (time‑zone errors)  
9.6. Neglecting Proofreading (typos, inconsistent terminology)  
9.7. Failing to Tailor Narrative (copy‑pasting generic text)  
9.8. Overlooking Reviewer Priorities (missing “why this funder”)
